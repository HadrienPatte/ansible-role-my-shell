# Ansible Role: My Shell

An Ansible Role that installs shell utilities and deploy various dotfiles on Ubuntu 18.04.

## Requirements

None.

## Role Variables

None.

# Dependencies

None.

# Example Playbook

```yaml
- name: Install shell utilities and deploy various dotfiles
  hosts: all
  roles:
    - hadrienpatte.my-shell
```
